# DigiList - Online Ordering System

DigiList is a web-based e-commerce platform specifically designed for ordering computer parts and components. The system features a smooth customer purchasing experience complete with automated email notifications, alongside a robust administrative dashboard equipped with sales analytics and inventory management.

## Features

### Customer Side
* **Product Catalog:** Browse and search through available PC parts and components.
* **Online Ordering:** Add items to a shopping cart and place orders seamlessly.
* **Email Notifications:** Automated email updates and order confirmations sent directly to the user's inbox.
* **Account Management:** Secure user registration, login, and self-service password change functionality.

### Admin Side
* **Sales Analytics:** Dynamic monitoring of sales performance, revenue trends, and ordering metrics.
* **Inventory & Stock Management:** Real-time tracking of stock levels with tools to update, add, or restock PC parts.
* **Order Management:** View, process, and update the status of customer orders.

## Tech Stack

* **Frontend:** HTML5, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Hosting Environment:** InfinityFree (Live Production) / XAMPP or WAMP (Local Development)
